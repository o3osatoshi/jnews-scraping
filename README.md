# jnews-scraping
Scraping all back issue articles from the jnews site

## Quick Start

1. Install the required packages:
```bash
poetry install
```

2. Set the environment variables:
```bash
cp .env.template .env 
vim .env
```

3. Run the script:
```bash
poetry run python main.py
```
